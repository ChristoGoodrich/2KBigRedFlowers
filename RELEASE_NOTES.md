# 2KBigRedFlowers v1.0.3 Android Preview

## 简体中文

这是面向 Android 侧载测试的移动端信息层级更新。

### 主要变化

- 移除移动端顶部的生涯总览条，页面打开后直接显示当前内容。
- 不再要求左右滑动查看摘要信息，减少首屏干扰。
- 保留桌面端完整的生涯总览展示。
- 保留底部 Tab 栏与“更多”抽屉中的设置入口。

### 下载内容

- `2KBigRedFlowers-1.0.3-debug.apk`
- `SHA256SUMS.txt`

Android APK 使用 debug 证书，仅用于侧载测试。

## English

This Android sideload preview simplifies the mobile information hierarchy.

### Highlights

- Removed the mobile career overview strip so each page opens directly on its
  primary content.
- Removed the horizontal swipe requirement for summary information on narrow
  screens.
- Preserved the full career overview strip on desktop screens.
- Preserved the bottom tab bar and Settings access through the More drawer.

### Downloads

- `2KBigRedFlowers-1.0.3-debug.apk`
- `SHA256SUMS.txt`

The Android APK uses a debug certificate and is intended for sideload testing.
