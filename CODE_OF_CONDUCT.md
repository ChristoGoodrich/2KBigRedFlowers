# Code Of Conduct / 行为准则

## Our Standards / 基本要求

Participants are expected to:

- Communicate respectfully and focus feedback on the work.
- Welcome constructive questions and different levels of experience.
- Avoid harassment, threats, discrimination, personal attacks, and publication
  of another person's private information.
- Keep issue discussions and pull request reviews relevant to the project.

参与本项目时，请遵守以下要求：

- 友善沟通，只针对具体工作提供反馈。
- 欢迎建设性问题，尊重不同经验水平的参与者。
- 禁止骚扰、威胁、歧视、人身攻击或公开他人隐私信息。
- Issue 讨论和 Pull Request 审核应围绕项目本身展开。

## Enforcement / 执行

Maintainers may edit, hide, or remove comments, issues, pull requests, or other
contributions that violate these standards. Repeated or severe violations may
result in a temporary or permanent ban from project spaces.

维护者可以编辑、隐藏或删除违反上述要求的评论、Issue、Pull Request 或其他内容。
重复或严重违规可能导致暂时或永久禁止参与项目。

For a confidential report, use the
[private reporting form](https://github.com/ChristoGoodrich/2KBigRedFlowers/security/advisories/new)
and state clearly that the report concerns conduct.

如需私密报告，请使用
[私密报告表单](https://github.com/ChristoGoodrich/2KBigRedFlowers/security/advisories/new)，
并明确说明报告内容与行为准则有关。
