# Security Policy / 安全策略

## Supported Versions / 支持版本

| Version / 版本 | Supported / 支持状态 |
| --- | --- |
| `1.0.x Preview` | Yes / 是 |
| Earlier versions / 更早版本 | No / 否 |

## Reporting A Vulnerability / 报告漏洞

Please do not open a public issue for a suspected vulnerability or include
credentials, Supabase keys, signing certificates, keystores, or personal data
in a report.

请勿通过公开 Issue 报告疑似漏洞，也请勿在报告中包含账号凭据、Supabase key、
签名证书、keystore 或个人数据。

Use GitHub's
[private vulnerability reporting form](https://github.com/ChristoGoodrich/2KBigRedFlowers/security/advisories/new).
Include the affected version, platform, reproduction steps, and impact. You
should receive an initial response within 7 days.

请使用 GitHub
[私密漏洞报告表单](https://github.com/ChristoGoodrich/2KBigRedFlowers/security/advisories/new)，
并提供受影响版本、平台、复现步骤和影响范围。通常会在 7 天内收到初步回复。
