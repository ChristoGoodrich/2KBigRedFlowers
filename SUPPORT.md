# Support / 支持

## Setup And Usage Questions / 配置与使用问题

Before opening an issue:

1. Read the [English README](./README.md) or [中文 README](./README.zh-CN.md).
2. Check [`PACKAGING.md`](./PACKAGING.md) for platform-specific build notes.
3. Search [existing issues](https://github.com/ChristoGoodrich/2KBigRedFlowers/issues).
4. Use the [support request form](https://github.com/ChristoGoodrich/2KBigRedFlowers/issues/new?template=support_request.yml)
   if the problem remains.

提交 Issue 前：

1. 阅读 [English README](./README.md) 或 [中文 README](./README.zh-CN.md)。
2. 在 [`PACKAGING.md`](./PACKAGING.md) 中检查各平台构建说明。
3. 搜索 [已有 Issue](https://github.com/ChristoGoodrich/2KBigRedFlowers/issues)。
4. 如果问题仍未解决，请使用
   [支持请求表单](https://github.com/ChristoGoodrich/2KBigRedFlowers/issues/new?template=support_request.yml)。

## Cloud Sync Checklist / 云同步检查清单

- Confirm that `supabase-cloud-sync.sql` has been run in the active Supabase
  project.
- Enter only the project URL and public anon key.
- Sign in with the same account on every device.
- Keep an exported local backup before clearing browser or app data.

- 确认已经在当前 Supabase 项目中执行 `supabase-cloud-sync.sql`。
- 只输入项目 URL 和公开 anon key。
- 在所有设备上登录同一个账号。
- 清理浏览器或应用数据前，请先导出本地备份。
