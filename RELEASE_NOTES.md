# 2KBigRedFlowers v1.0.2 Android Preview

## 简体中文

这是面向 Android 侧载测试的移动端导航更新。

### 主要变化

- 移除移动端固定品牌顶栏，让内容直接从屏幕顶部开始。
- 新增底部 Tab 栏：总览、建模、球员、比赛、更多。
- 将质量、对比和设置收进“更多”抽屉。
- 保留备份、主题、语言与 OCR 设置入口。
- 修复浅色主题 Android WebView 中设置抽屉的定位问题。

### 下载内容

- `2KBigRedFlowers-1.0.2-debug.apk`
- `SHA256SUMS.txt`

Android APK 使用 debug 证书，仅用于侧载测试。

## English

This Android sideload preview updates the mobile app navigation.

### Highlights

- Removed the fixed mobile brand header so content starts at the top of the
  screen.
- Added a five-item bottom tab bar: Overview, Builds, Players, Games, and More.
- Moved Quality, Compare, and Settings into the More drawer.
- Preserved access to Backup, Theme, Language, and OCR Settings.
- Fixed Settings drawer placement in the light-theme Android WebView.

### Downloads

- `2KBigRedFlowers-1.0.2-debug.apk`
- `SHA256SUMS.txt`

The Android APK uses a debug certificate and is intended for sideload testing.
