# 2KBigRedFlowers v1.0.4 Android Preview

## 简体中文

这是面向 Android 侧载测试的移动端浅色模式与账号中心更新。

### 主要变化

- 完全重构移动端浅色模式，使用高对比白底、单列内容与原生 App 风格底部 Tab 栏。
- 新增移动端动态页标题，移除桌面端信息层级在窄屏上的干扰。
- 重排首次使用页：生涯摘要、主要操作、使用步骤一眼可见。
- 将比赛列表在移动端改为卡片式记录，避免横向表格阅读。
- 新增独立账号中心，登录、注册、云同步状态和恢复工具不再藏在数据备份里。
- 修复移动端弹窗被底部 Tab 栏遮挡的问题。

### 下载内容

- `2KBigRedFlowers-1.0.4-debug.apk`
- `SHA256SUMS.txt`

Android APK 使用 debug 证书，仅用于侧载测试。

## English

This Android sideload preview rebuilds the mobile light-mode experience and
adds a standalone Account Center.

### Highlights

- Rebuilt mobile light mode with high-contrast surfaces, single-column content,
  and a native-style bottom tab bar.
- Added compact mobile page titles and removed desktop hierarchy from narrow
  screens.
- Reworked the first-run view into summary, action, and onboarding sections.
- Converted mobile game records into readable card rows.
- Added a standalone Account Center for authentication, sync state, cloud
  configuration, and recovery tools.
- Kept modal sheets above the mobile bottom tabs.

### Downloads
